title: Application
description: A guide to using the deployit CLI to manage applications
---


The deployit CLI can be used to create, deploy, and manage applications.

## Requirements

Is required to be already authenticated with the deployit CLI.


## Commands

1. `deployit app create` - Create a new application.
2. `deployit app delete` - Delete an application.
3. `deployit app deploy` - Deploy an application.
4. `deployit app stop` - Stop a running application.
