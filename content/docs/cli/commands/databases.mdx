title: Databases
description: A guide to using the deployit CLI to manage databases
---

The deployit CLI can be used to create, deploy, and manage databases.

## Requirements

Is required to be already authenticated with the deployit CLI.


## Commands

### MariaDB 

1. `deployit database mariadb create` - Create a new mariadb database.
2. `deployit database mariadb delete` - Delete an mariadb database.
3. `deployit database mariadb deploy` - Deploy a mariadb database.
4. `deployit database mariadb stop` - Stop a running mariadb database.
