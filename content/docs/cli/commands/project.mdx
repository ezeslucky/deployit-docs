description: A guide to using the deployit CLI to manage projects
---


The deployit CLI can be used to create, deploy, and manage projects.

## Requirements

Is required to be already authenticated with the deployit CLI.

## Commands

1. `deployit project create` - Create a new project.
2. `deployit project info` - Get information about a project.
3. `deployit project list` - List all projects.
