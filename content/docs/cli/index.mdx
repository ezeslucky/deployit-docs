title: Introduction
description: A guide to using the deployit command-line interface
---

deployit CLI is a command-line tool for remotely managing your deployit server. It simplifies creating, deploying, and managing applications and databases.

## Installation

```bash
npm install -g @deployit/cli
```

## Usage

```bash
deployit COMMAND
```
To get help on a specific command:

```bash
deployit COMMAND --help
```
