## 卸载步骤

按照以下步骤从您的服务器中完全删除 deployit 及其组件。

### 第 1 步： 删除 Docker Swarm 服务

首先，删除 Docker Swarm 中的 deployit 以及相关服务:

```bash
docker service rm deployit deployit-traefik deployit-postgres deployit-redis
```

### 第 2 步： 删除 Docker 挂载卷

接下来，删除 deployit 创建的 Docker 卷：

```bash
docker volume rm -f deployit-postgres-database redis-data-volume
```

### 第 3 步： 删除 deployit 文件

最后，从服务器上删除 deployit 文件和目录：

```bash
sudo rm -rf /etc/deployit
```
