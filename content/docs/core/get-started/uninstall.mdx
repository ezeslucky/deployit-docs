title: Uninstall
description: "Learn how to uninstall deployit on your server"
---

import { Step, Steps } from 'fumadocs-ui/components/steps';

Follow these steps to completely remove deployit and its components from your server.

<Steps>
<Step>
Remove the docker swarm services created by deployit:

   ```bash
   docker service rm deployit deployit-traefik deployit-postgres deployit-redis
   ```
 
</Step>
 
<Step>
 
Remove the docker volumes created by deployit:

   ```bash
   docker volume rm -f deployit-postgres-database redis-data-volume
   ```
 
</Step>
 
<Step>

Remove the deployit files and directories from your server:

   ```bash
   sudo rm -rf /etc/deployit
   ```
 
</Step>
</Steps>
